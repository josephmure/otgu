# otgu module

Examples
  Template examples can be found here: C:\Python27\Lib\site-packages\otgu\examples\otgu

Installation
  Silent install can be done with the switch /S (e.g. openturns-otgu-0.0-py27.exe /S).
  To test otgu module installation, in an OpenTURNS python console type:
    import otgu
    print otgu.__version__

Uninstallation
  OTGU module will be automatically uninstalled at OpenTURNS removal.
  OTGU module can be removed from OpenTURNS with uninst-template.exe uninstaller.
  Silent uninstall can be done with the switch /S.
